== Frequently Asked Questions ==

**Is TotalForm a self-hosted plugin?**

Yes, TotalForm is a self-hosted solution, you have the full control over the data.

**Are there any quotas or limitations on features?**

There is no feature limit, you can have as many forms and entries as you need.

**What methods can I use to integrate a form?**

You can integrate forms using shortcodes, REST API, or PHP functions.

**What I need to run TotalForm?**

You'll need WordPress 5.0+, PHP 7.0+, and MySQL 5.7+
